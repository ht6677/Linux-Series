# Funny Terminals

```sh
$ sudo apt-get install cowsay
```

在终端中输入 cowsay hello 就会出现一只牛对你说 hello 了。如果你想要其他动物，那么可以使用 cowsay -l 命令就能看见有什么动物可以给你选了。比如我想让猫咪对我 hello ，可以这样写：cowsay -f meow hello
